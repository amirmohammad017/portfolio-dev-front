## 👨‍💻 About This Project

A modern, responsive, and server-side rendered **personal portfolio website** built with React and Vite — showcasing projects, blog posts, and contact information.

---

## ✨ Features

- 🎨 Clean and modern UI design
- 📱 Fully responsive across all devices
- ⚡ Fast performance with Vite
- 🖥️ SSR (Server-Side Rendering) support
- 📝 Blog section with detailed post view
- 🗂️ Projects showcase
- 📬 Contact page
- 🐳 Docker support for easy deployment

---

## 🚀 Getting Started
Prerequisites
Node.js >= 18
npm or yarn
Installation
bash

##📥 Clone the repository
git clone https://github.com/amirmohammad017/portfolio-dev-front.git

##▶️ Run development server
